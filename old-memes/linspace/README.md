# linspace

An implementation of MATLAB's linspace function.

## install

    npm install linspace

## require

    > var linspace = require('linspace');

## use

### linspace(from, to, length)

Acts like linspace from matlab:

    > linspace(1,5,15)
    [ 1,

      4.1428571428571415,
