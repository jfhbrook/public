# logspace

An implementation of MATLAB's logspace in javascript.

## install

    npm install logspace

## require

    > var logspace = require('logspace');

## use

### logspace(logFrom, logTo, length)

Acts like logspace from matlab:

    > logspace(-1,6,10)
    [ 0.1,
      0.599484250318941,
      3.5938136638046276,
      21.544346900318846,
      129.1549665014884,
      774.263682681127,
      4641.588833612777,
      27825.59402207126,
      166810.0537200059,
      1000000 ]
